CoderDojo松原とは

CoderDojoは世界中に広がる子ども向けの非営利プログラミング道場です。
CoderDojo松原は「みんなで楽しくアウトプットしていく」をコンセプトに、大阪府松原市で活動しています。

無料で参加できる

参加費は一切かかりません。CoderDojoの理念のもと、誰でも気軽に参加できます。

自分のペースで学べる

やりたいことを自分で決めて取り組む「自主学習」の場です。メンターがサポートします。

仲間と一緒に楽しめる

同じ興味を持つ仲間と交流しながら、創作活動を楽しみましょう。

こんなことができます

初心者 Scratch
初心者 Minecraft
経験者 Ruby
経験者 Python
経験者 JavaScript
その他 レジン制作・季節イベントも！

現役Webプログラマのメンターがサポートします。わからないことは何でも聞いてください。

注意事項

開催判断について

子どもたちの参加人数によっては、開催を見送る場合がございます。当日11時時点の状況で判断いたしますので、お出かけ前にLINEに連絡が来ていないかご確認をお願いします。

撮影について

イベントレポートおよびSNS用に、プログラミング中の様子を撮影しています。顔出しNGの場合は申し込み時のチェックボタンにてお知らせください。

場のルールについて

CoderDojo松原は何かを制作する場です。参考のためにサイトをチェックする、作品に触れる、動画を閲覧するのは問題ありません。ただし、メンターが注意する場合は指示に従うようお願いいたします。指示に従えない場合、他の参加者にも申し訳ないのでイベント時間に関わらずお帰りいただきます。

運営について

CoderDojo松原はあくまで一般の有志メンバーで成り立っています。

CoderDojoの理念から参加費の徴収は行っておりませんが、活動を継続させるための寄付を任意でお願いしています。なお、皆様からの寄付は会場費や備品代に使わせていただきます。

皆様のご理解・ご協力のほど宜しくお願いします。
